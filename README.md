# Useful Algorithms for Reuse
This repository holds a collection of small and broad, but very useful, algorithms that are suitable for a wide variety of programming uses. They can be implemented in other programs with a little rework. Anyone is free to use the algorithms I post here. Any sources I used to help make these algorithms will be linked at the bottom of the file of said algorithm.
